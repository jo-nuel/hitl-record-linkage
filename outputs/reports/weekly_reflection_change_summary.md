# Weekly Reflection Change Summary

After feedback, we removed the Synthea-based duplicate generation pipeline because it weakened evaluation validity. We moved to FEBRL as the primary dataset because it provides benchmark linkage data with ground-truth links. We also changed the matching design toward an EMPI-inspired workflow using multi-pass blocking, field-level evidence, probabilistic or hybrid scoring, threshold tuning, and human review of uncertain pairs. This made the project more realistic, more defensible, and better aligned with healthcare patient matching systems.
